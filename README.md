# KM Blog - Multiuser blog
Simple multi user blog app created using nestjs and reactjs.

## Features
- Login / Register
- Create / Upate / Delete post for login user
- Comment a post
- Browse post by category
- Browse post by user
- Responsiveness

## Setup
To run this project, install it locally using yarn:

```
$ cd ../backend
$ yarn install
$ yarn run start:dev
$ cd ../frontend
$ yarn install
$ yarn start
```
